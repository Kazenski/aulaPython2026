O traçado do movimento do robô é mostrado a seguir: CIMA 5 PARA BAIXO 3 PARA A ESQUERDA 3 PARA A DIREITA 2. Os números após a direção representam os passos. Escreva um programa para calcular a distância entre a posição atual e o ponto inicial após uma sequência de movimentos. Se a distância for um número de ponto flutuante, imprima o inteiro mais próximo. Exemplo: Se as seguintes tuplas forem fornecidas como entrada para o programa: CIMA 5 PARA BAIXO 3 PARA A ESQUERDA 3 PARA A DIREITA 2, então a saída do programa deverá ser: 2

Solução:

import math
pos = [0,0]
while True:
    s = input()
    if not s:
        break
    movement = s.split(" ")
    direction = movement[0]
    steps = int(movement[1])
    if direction=="UP":
        pos[0]+=steps
    elif direction=="DOWN":
        pos[0]-=steps
    elif direction=="LEFT":
        pos[1]-=steps
    elif direction=="RIGHT":
        pos[1]+=steps
    else:
        pass

words.sort()

for w in words:
    print("%s:%d" % (w,freq[w]))

import math
def bin_search(li, element):
    bottom = 0
    top = len(li)-1
    index = -1
    while top>=bottom and index==-1:
        mid = int(math.floor((top+bottom)/2.0))
        if li[mid]==element:

        elif li[mid]>element:
            top = mid-1
        else:
            bottom = mid+1

    return index

Solução:

def removeDuplicate( li ):
    newli=[]
    seen = set()
    for item in li:
        if item not in seen:
            seen.add( item )
            newli.append(item)

    return newli

li=[12,24,35,24,88,120,155,88,120,155]
print(removeDuplicate(li))

Questão 100
Escreva um programa para resolver um enigma clássico da China antiga: Contamos 35 cabeças e 94 patas entre as galinhas e os coelhos em uma fazenda. Quantos coelhos e quantas galinhas temos?

Dica: Use o laço `for` para iterar por todas as soluções possíveis.

Solução:

def solve(numheads,numlegs):
    ns='No solutions!'
    for i in range(numheads+1):
        j=numheads-i
        if 2*i+4*j==numlegs:
            return i,j
    return ns,ns

numheads=35
numlegs=94
solutions=solve(numheads,numlegs)
print(solutions)
